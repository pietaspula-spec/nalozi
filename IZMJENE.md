# Izmjene

## Zadnje pripremljene izmjene

### 1. Help datoteka

- pripremljen novi kompletni `help.html`
- uključeni tabovi `Nalozi`, `Obračun` i `Izvještaji`
- uključene postavke, realtime i backup funkcije

### 2. Logout i lokalni podaci

- ručna odjava sada briše i lokalne obračune, naknade i porezne postavke
- brišu se i lokalni `nseen_*` markeri ažuriranja naloga
- ponašanje je sigurnije za dijeljene uređaje i više admin korisnika

### 3. Admin provjera pri startu

- mrežne greške pri početnoj admin provjeri više se ne ignoriraju
- aplikacija sada ostaje na login ekranu ako provjera ne uspije
- korisnik dobije jasniju poruku umjesto poluulogiranog stanja

### 4. README

- README je proširen tako da odgovara stvarnoj `v3` verziji aplikacije
- dodan opis obračuna, izvještaja i settings drawera

### 5. Paket za upload

Za upload u repo pripremljene su ove datoteke:

- `index.html`
- `help.html`
- `sw-nalozi.js`
- `README.md`
- `IZMJENE.md`
